这是一个连连看游戏，正处于开发阶段，功能暂不完善，欢迎体验
v3.1的bug：
错误：_T( ) 宏不能用
位置：所有 setfont、outtextxy 里的 _T("宋体")、_T("本关通关")
原因：Dev-C++ 不支持 _T() 宏，只有 VS 能用
正确写法：直接写字符串 "宋体"、"本关通关"
2. 错误：kbhit() / getch() 未定义
位置：主循环、通关循环
原因：Dev-C++ 需要 下划线版本
正确写法：_kbhit()、_getch()
3. 错误：宽字符字符串不兼容
位置：outtextxy 中的中文
原因：Dev-C++ 默认使用多字节字符集，不能加 L 或 _T
正确写法：直接 "中文"
4. 错误：鼠标监听没有 MouseHit() 判断
位置：主循环鼠标部分
原因：不判断会导致程序卡死、疯狂占用 CPU
正确写法：必须先写 if(MouseHit())
5. 错误：通关后没有重置选中状态
位置：进入下一关时
后果：下一关自动选中格子
修复：selected_r = -1; selected_c = -1;
